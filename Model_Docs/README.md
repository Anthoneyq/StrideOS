# STRIDE OS + Running Performance Markdown Package

This package contains the Markdown files from our STRIDE OS and running-performance / race-prediction work.

## Included files

1. `01-stride-os-predictive-model-validation-and-time-decay-research.md`
   - Full research report on recency weighting, time decay, athlete archetypes, event correlation, and confidence modeling.

2. `02-detraining-retention-pr-recency-weighting.md`
   - Focused research notes on detraining, physiological retention, and how old PRs should decay by athlete type and event family.

3. `03-product-spec-prd.md`
   - STRIDE OS product requirements and behavior goals.

4. `04-technical-model-spec.md`
   - Technical model design, formulas, weighting logic, and pseudocode.

5. `05-coach-facing-explanation.md`
   - Plain-language explanation for coaches and athlete-facing UX copy.

6. `06-scientific-research-summary.md`
   - Scientific synthesis and implementation recommendation with citations.
